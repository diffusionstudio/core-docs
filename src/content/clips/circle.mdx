# Circle Clip

The `CircleClip` class inherits from the `ShapeClip` and is almost identical to the `RectangleClip`.

## Basic Usage

```typescript
import * as core from '@diffusionstudio/core';

const circle = new core.CircleClip({
  position: 'center',
  width: '50%',
  height: '50%',
  fill: '#FF0000',
});
```
